# APS Online - Associação Paranaense de Suinocultores

Site institucional modernizado da APS, com design premium e foco em representatividade.

## Tecnologias Utilizadas
- HTML5 / CSS3 (Vanilla)
- Javascript
- Glassmorphism & UI Moderno
- Otimizado para Vercel

## Como rodar localmente
Basta abrir o arquivo `index.html` em qualquer navegador.
Ou usar o comando: `npm start` (requer Node.js instalado).

## Deploy
O projeto está configurado para deploy automático na Vercel via arquivo `vercel.json`.
